The snowclone is a kind of phrasal template since it comes with one or more empty "slots" that get filled with words to create a new phrase. Some examples:

  • I'm not an X, but I play one on TV
  • In X, no one can hear you Y
  • X and Y and Z, oh my!

The Mozilla Developer Network has this to say about the style attribute:

Note that it is recommended for styles to be defined in a separate file or files. This attribute and the <style> element have mainly the purpose of allowing for quick styling, for example for testing purposes.
