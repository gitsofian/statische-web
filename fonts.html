Specifying the type-size Property

To use the type-size property, you must know the main measurement units used in CSS:
  • px — An absolute measurement equal to 1/96 of an inch
  • pt — An absolute measurement equal to 1/72 of an inch
  • em — A relative measurement equal to the element's default, inherited, or defined font size
  • rem — A relative measurement equal to the font size of the root element of the web page
  • vw — A relative measurement equal to 1/100 of current width of the browser window
  • vh — A relative measurement equal to 1/100 of current height of the browser window
